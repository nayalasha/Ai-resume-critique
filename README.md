# 🤖 AI Resume Critique

An AI-powered resume analysis application built with Python and CustomTkinter that helps users improve their resumes through smart feedback, scoring, and professional recommendations.

---

## ✨ Features

- 📄 Paste and analyze resume content
- 📊 Resume scoring system
- 🤖 AI-generated feedback and suggestions
- 🔍 Keyword and skills analysis
- 📏 Resume length and structure checks
- 💼 Professional resume recommendations
- 🎨 Modern desktop interface built with CustomTkinter
- 📁 Future support for PDF resume uploads

---

## 🛠 Technologies Used

- Python
- CustomTkinter
- OpenAI API *(planned)*
- PyPDF2 *(planned)*
- Git & GitHub
