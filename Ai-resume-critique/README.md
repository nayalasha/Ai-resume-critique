# 🤖 AI Resume Critique

An AI-powered resume analysis application built with Python that helps users improve their resumes with smart feedback, resume scoring, and professional suggestions.

---

## ✨ Features

* 📄 Resume analysis
* 🤖 AI-generated feedback
* 📊 Resume scoring system
* 🔍 Keyword checker
* 🧠 Skills section detection
* 📏 Word count analysis
* 💡 Professional improvement suggestions
* 🎨 Clean and beginner-friendly UI

---

## 🛠 Technologies Used

* Python
* Streamlit
* OpenAI API
* Pandas
* PyPDF2

---

## 🚀 How It Works

1. User uploads or pastes a resume
2. The app analyzes the resume text
3. AI generates suggestions and feedback
4. A resume score is calculated
5. Users improve their resumes using the recommendations

---

## 📂 Installation

Clone the repository:

```bash
git clone https://github.com/nayalasha/Ai-resume-critique.git
```

Go into the project folder:

```bash
cd ai-resume-critique
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
